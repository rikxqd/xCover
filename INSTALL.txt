xCover - Install
===================

Updated:    1st November 2015



xCover requires manual installation, by copying files after building,
as described in the accompanying doc/html/0.3.3/index.html file.


NOTE: xCover depends on the (header-only) STLSoft libraries - version
  1.9.122 or later - available from http://stlsoft.org/.
